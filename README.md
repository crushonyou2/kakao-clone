# Kakao Clone

카카오톡 페이지를 클론한 정적 웹 프로젝트입니다.
HTML과 CSS만으로 실제 카카오 웹 로그인 화면과 유사하게 구성하되, 왓챠피디아의 UI를 따와 차별점을 두었습니다.  
반응형 디자인을 고려하여 다양한 화면에서도 자연스럽게 동작합니다.
